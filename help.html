ANOVA Univariado (one-way)

Compara as médias de dois ou mais grupos para uma única variável resposta.

Quando usar

  • Você tem uma variável resposta contínua (ex: resistência, rendimento).
  • Você tem um fator categórico com ≥ 2 níveis/grupos (ex: tipo de catalisador A, B, C).
  • As observações são independentes dentro e entre grupos.
  • Os resíduos seguem distribuição normal com variâncias iguais (homoscedasticidade).

Modelo e equações

O modelo estatístico é:

y_ij = μ + τ_i + ε_ij y_ij = j-ésima observação do i-ésimo grupo μ = média geral τ_i = efeito do i-ésimo tratamento (Σ n_i·τ_i = 0) ε_ij ~ N(0, σ²) independentes

A variação total é particionada em:

SQT = SQTr + SQE SQTr = Σ_i n_i · (ȳ_i. − ȳ..)² GL = a − 1 [entre grupos] SQE = Σ_i Σ_j (y_ij − ȳ_i.)² GL = N − a [dentro grupos / erro] SQT = Σ_i Σ_j (y_ij − ȳ..)² GL = N − 1 [total] QMTr = SQTr / (a − 1) QME = SQE / (N − a) F = QMTr / QME ~ F(a−1, N−a) sob H₀: τ₁=τ₂=...=τₐ=0

Se F > F_crítico (ou p-valor < 0,05), rejeita-se H₀ — pelo menos um grupo difere.

O post-hoc de Tukey (HSD) identifica quais pares diferem:

HSD = q_{α, a, ν} · √(QME / n) q = quantil da distribuição Studentized Range ν = GL_E = N − a n = tamanho do grupo (balanceado)

Premissas e diagnósticos

  • Normalidade: Shapiro-Wilk nos resíduos (p > 0,05 = OK).
  • Homoscedasticidade: Teste de Levene ou Bartlett (p > 0,05 = OK).
  • Independência: garantida pelo delineamento (aleatorização).
  • Se violadas: usar Kruskal-Wallis (não-paramétrico) ou transformação da resposta.

ANOVA Multivariado (MANOVA)

Compara vetores de médias de grupos para duas ou mais variáveis resposta simultaneamente.

Quando usar

  • Você tem duas ou mais variáveis resposta correlacionadas (ex: resistência + dureza).
  • Analisar cada resposta separadamente inflaria o erro tipo I — a MANOVA controla isso.
  • Os erros seguem distribuição normal multivariada com mesma matriz de covariância Σ.

Modelo e critérios

Modelo: y_ij = μ + τ_i + ε_ij (vetores em ℝ^p) Matrizes SSCP: H = Σ_i n_i·(ȳ_i − ȳ)(ȳ_i − ȳ)' GL_H = a − 1 [hipótese] E = Σ_i Σ_j (y_ij−ȳ_i)(y_ij−ȳ_i)' GL_E = N − a [erro] Critérios (em ordem de robustez): Wilks' Λ = |E| / |H+E| → F de Rao Traço Pillai = tr[H(H+E)⁻¹] Hotelling T² = tr[E⁻¹H] Roy θ_max = maior autovalor de (H+E)⁻¹H

Planejamento Fatorial 2k

Estuda k fatores em 2 níveis cada, avaliando efeitos principais e interações com 2^k corridas.

Quando usar

  • Você quer identificar quais fatores influenciam a resposta e se há interações.
  • Cada fator tem exatamente 2 níveis: baixo (−1) e alto (+1).
  • Muito eficiente: 2k corridas cobrem todo o espaço experimental de k fatores.
  • Com réplicas, é possível estimar o erro puro e testar significância.

Codificação e contrastes de Yates

Codificação: x_i = (X_i − X̄_i) / (ΔX_i / 2) x_i ∈ {−1, +1} Contraste de Yates para o efeito i: Contraste_i = Σ_l c_{li} · ȳ_l (c_{li} ∈ {−1, +1}) Efeito estimado: Efeito_i = Contraste_i / (N/2) = Contraste_i · 2 / N Soma de quadrados (exata): SQ_i = (Contraste_i)² / N GL = 1 para cada efeito

Com n réplicas por ponto, o erro puro tem GL = 2^k·(n−1).

Sem réplicas (n=1), usa-se o half-normal plot ou interações de alta ordem como estimativa de σ².

Planejamento Composto Central (CCD)

Permite ajustar um modelo quadrático completo para otimizar a resposta (superfície de resposta — RSM).

Quando usar

  • Após um 2k indicar quais fatores são importantes, o CCD localiza o ponto ótimo.
  • Necessário quando a resposta tem curvatura (máximo/mínimo) na região experimental.
  • Combina: pontos fatoriais (±1), axiais (±α) e centrais (0).

Estrutura e modelo

Alpha rotável: α = (2^k)^(1/4) k=2 → α=1,414 | k=3 → α=1,682 | k=4 → α=2,000 Número de pontos: Fatoriais: 2^k | Axiais: 2k | Centrais: n₀ (≥ 3) Modelo quadrático completo (k=2): ŷ = β₀ + β₁x₁ + β₂x₂ + β₁₁x₁² + β₂₂x₂² + β₁₂x₁x₂ Estimação: β̂ = (X'X)⁻¹ · X'y Ponto estacionário: ∂ŷ/∂x = 0 → x_s = −½ B⁻¹ b b = vetor de β lineares, B = matriz de β quadráticos e cruzados

Cuidados

  • O teste de Lack of Fit deve ser não-significativo (p > 0,05) para o modelo quadrático ser válido.
  • Se o ponto ótimo estiver fora da região experimental, o resultado é uma extrapolação — execute novos experimentos.
  • Use pelo menos 3 pontos centrais para estimar a curvatura pura e o erro puro.
